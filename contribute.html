Requests and Contributions¶

The primary vision for pyfluo is a stable and minimalistic tool set that is broadly applicable. The package is still in its infancy, and features will be added according to demand as time progresses.

To request features¶

Send me an email.

To contribute¶

Contributions to this package are gratefully welcomed. If you have succeeded in implementing a feature that you see fit for this package, and you are willing to share it, send me an email, and I will consider integrating it into the package.
